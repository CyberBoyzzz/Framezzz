# Framezzz
there are some frames, you know?

Let's looks at some cool comics.

## Project structure

- framezzz-api (core data product experience service)
- framezzz-auth (customer authorisation service)
- framezzz-frontend (client side service)

## How to run project?

```shell
docker compose up
```
